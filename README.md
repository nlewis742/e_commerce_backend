# E Commerce Backend

## Description
This is a backend for an e-commerce site. It uses Express.js API and Sequelize to interact with a MySQL database.

## Installation Requirements
* Node.js
* MySQL2
* Sequelize
* Express.js
* dotenv

And Insomnia Core to test the routes.
